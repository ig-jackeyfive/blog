## 闭包

### 什么是闭包

闭包是指有权访问另外一个函数作用域中的变量的函数

### 闭包产生的原因?

作用域链...

闭包产生的本质就是，当前环境中存在指向父级作用域的引用。

### 闭包有哪些表现形式?

1. 返回一个函数
2. 函数作为参数传递
3. 回调函数
4. IIFE

### 如何解决下面的循环输出问题？

```javascript
for(var i = 1; i <= 5; i ++){
  setTimeout(function timer(){
    console.log(i)
  }, 0)
}
```

1. IIFE
2. 给定时器传入第三个参数，作为timer函数的参数
3. let块级作用域
